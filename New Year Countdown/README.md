# New Year Countdown Timer

A responsive New Year Countdown Timer built with Vanilla JavaScript, HTML, and CSS.
It dynamically counts down the remaining days, hours, minutes, and seconds until the start of the next year.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript (Date, setInterval, DOM manipulation)
