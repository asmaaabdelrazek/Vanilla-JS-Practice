# Frontend Practice Tasks

This repository contains multiple small frontend projects and practice tasks built with HTML5, CSS3, and Vanilla JavaScript.

Each folder represents an independent mini project or a UI component that could be part of a larger real-world project. 
The goal is to practice and improve frontend skills by building real-world website sections, interactive features, and UI components.

## Purpose
The main goal of this repository is to practice and improve my frontend skills by building individual UI components and interactive features. 
These tasks focus on understanding the structure (HTML), styling (CSS), and behavior (JavaScript) of real-world website sections.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, events, logic)
